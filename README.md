# Video-Group-Meeting

Videe-Group-Meeting is a simple video chat application for multi-users based on React, Node Express and WebRTC.

**Technologies Used**

- React
- Node + Express
- WebRTC
- Socket.io


---

## Features

- Join a room
- Video Streaming
- Text chat
- Mute Video/Audio
- Screen Sharing
